# 🎮 遊戲樂園

用身體玩的網頁體感遊戲（手機鏡頭 AI 偵測 + 投放電視，類似 Kinect）。

**▶️ 開始玩：** https://vinbcent.github.io/kids-games/

## 資料夾結構

```
├── index.html        遊戲選單（網站首頁）
├── 共用/              共用資源（PWA 設定、App 圖示）
└── 遊戲N-名稱/        每款遊戲一個資料夾，互相獨立
    ├── index.html    遊戲本體（單一檔案）
    └── README.md     玩法與設置說明
```

## 遊戲列表

| # | 遊戲 | 玩法 |
|---|------|------|
| 1 | [🚀 火箭冒險](遊戲1-火箭冒險/) | 身體左右移動控制火箭、雙手舉高加速衝刺、收星星躲炸彈 |
| 2 | [🚛 水泥車大作戰](遊戲2-水泥車大作戰/) | 身體左右移動開水泥車、雙手舉高倒水泥，倒滿貨車、黏住小汽車 |

## 技術

- 純 HTML/Canvas/Web Audio，零依賴、零安裝
- 人體偵測：MediaPipe Pose Landmarker（全部在裝置本機運算，影像不上傳）
- PWA：加到主畫面即可全螢幕遊玩
