# 🚀 火箭冒險（遊戲 1）

體感火箭遊戲 — 像 Xbox Kinect 一樣，用身體控制火箭！

## 玩法

- **左右移動身體** → 火箭跟著左右飛
- **雙手舉高** → 🔥 衝刺加速！
- **收集 ⭐ 星星**：每 10 顆放煙火 + 語音稱讚，每 20 顆火箭換新顏色
- **🎁 禮物盒**：碰到會爆出驚喜
- 沒有失敗、沒有 Game Over，只有獎勵 ✨

## 電視設置步驟（用身體玩）

1. 手機和 Chromecast 連同一個 Wi-Fi
2. 打開手機的 **Google Home App** → 選擇 Chromecast → **投放我的螢幕**
3. 手機**橫放**在電視上方，**前鏡頭朝向沙發/玩的地方**
4. 在手機 Chrome 打開遊戲網址，按「🏃 用身體玩」
5. 允許鏡頭權限 → 小朋友站到電視前面約 1.5～3 公尺 → 開始！

### 小技巧

- 房間燈光要夠亮，鏡頭才看得清楚
- 左下角有鏡頭小視窗，綠點 = 偵測到的位置（舉手會變黃點）；確認沒問題可按 ✕ 關掉
- 火箭會「平滑跟隨」，稍微慢半拍是正常的（這樣投放延遲感覺不出來）
- 影像全部在手機上即時運算，**不會上傳到任何伺服器**

## 在電腦上測試

直接用 Chrome 打開 `index.html`，按「👆 用手指玩」即可（拖曳控制、兩指按住加速）。
有 webcam 的話「用身體玩」在電腦上也能跑。

## 技術備註

- 單一 HTML 檔，無需安裝
- 人體偵測：MediaPipe Pose Landmarker（CDN 載入，需要網路）
- ⚠️ 鏡頭功能需要 **HTTPS 網址或 localhost**（瀏覽器安全限制），手機上建議部署到 GitHub Pages 等免費空間
