# 🚛 水泥車大作戰（遊戲 2）

體感水泥車遊戲（橫式設計）— 開著水泥車在高架橋上，把水泥倒進下面的貨車！

## 玩法

- **左右移動身體** → 橋上的水泥車跟著左右開
- **雙手舉高** → 🌊 倒水泥！（水泥無限量，放心倒）
- **🛻 空貨車（慢慢開）**：把車斗倒滿 → 「叮！」滿載出發，計數 +1，每 5 台放煙火、每 10 台水泥車換新顏色
- **🚗 小汽車**：被水泥淋到會整台黏住——輪子空轉、狂按喇叭、整台車抖抖抖 😆 3 秒後「咻！」掙脫跑掉（純搞笑，不扣分；一台車只會被黏一次）
- 沒有 Game Over，沒有失敗 ✨
- **🛑 護眼提醒**：跑太靠近電視 → 警報音 + 遊戲強制暫停 5 秒（畫面倒數），退後才能繼續玩

## 手指模式

按住畫面就會**邊移動邊倒水泥**，放開就停（適合在電腦或平板上玩）。

## 📺 投影到電視

和火箭冒險相同：手機用 Google Home App「投放我的螢幕」到 Chromecast（**建議 5GHz Wi-Fi** 才順），手機橫放電視上方、鏡頭朝玩的人，選「🏃 用身體玩」。

## 技術備註

- 單一 HTML 檔，無需安裝；人體偵測 MediaPipe Pose Landmarker（影像只在裝置本機運算，不上傳）
- 鏡頭功能需要 HTTPS 網址或 localhost
- 本遊戲以**橫式**設計（電視/手機橫放），直拿手機會出現轉向提示
