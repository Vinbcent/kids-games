# 🚜 挖土機挖挖樂（遊戲 3）

體感挖土機遊戲（橫式設計）。**上下分層**的工地：挖土機在上方工地台、小汽車在下方馬路、馬路底下是**深地底剖面**。蹲下挖坑讓車掉進去，雙手高舉填土把車救出來；地底還埋著各種驚喜！😆

## 版面

```
        🚜  ← 挖土機（上方工地台，左右開）
━━━━━━━━━━━━━━
      ║ ← 手臂往下挖／往上填
 🚗→   ║   🚕     馬路（小汽車在下面跑）
━━━━━━━━━━━━━━
 ▽坑  💧  💎  🦴  🪨   深地底（埋著驚喜）
```

## 玩法

- **左右移動身體** → 挖土機在工地台上左右開
- **蹲下** → ⛏️ 鏟子往下挖坑（挖越久越深）
- **雙手高舉**（兩手舉過頭）→ 🪣 倒土填坑
- **🕳️ 挖坑讓車掉進去**：坑夠深，馬路上的車就會 **掉進去翻滾、輪子空轉、頭上冒❗、亂按喇叭** 😆 計數 +1，每 5 台放煙火、每 10 台挖土機換新顏色
- **🪣 填土把車救出來**：對著有車的坑填土 → 車被頂出來開心開走（救援 +1）。沒救牠也沒關係，車太久會自己爬出來（不會卡關）
- **地底四種驚喜**（沒挖到時會露半透明線索）：
  - 💧 **水管** → 挖到就「砰！」噴大水柱，**填土可以蓋住止水**（噴一陣子也會自己停）
  - 💎 **寶藏** → 噴金幣、加分、放煙火
  - 🦴 **恐龍骨** → 挖出一隻恐龍「Roar！」
  - 🪨 **挪不動大石** → 鏟子「鏗！」彈開、挖土機抖一下，這格挖不深（換地方挖）
- **路上有 6 種車，掉坑叫聲都不一樣**：🚗 小汽車「叭叭」、🚕 計程車「叭～叭～」、🚓 警車「咿─嗚─」紅藍燈狂閃、🚌 巴士低音氣笛「噗──」、🏎️ 跑車「嗶嗶嗶」（超快很難挖到！）
- 沒有 Game Over，沒有失敗 ✨

- **🛑 護眼提醒**：跑太靠近電視 → 警報音 + 遊戲強制暫停 5 秒（畫面倒數），退後才能繼續玩

## 手指模式

拖曳移動挖土機；**按住畫面下半=挖**、**按住畫面上半=填土**（適合在電腦或平板上玩）。

## 體感備註（蹲下 / 舉手偵測）

- 蹲下：用「大腿/軀幹」比例判定（高門檻 + 遲滯防抖）；鏡頭看不到腿時自動改用「肩膀相對站立基準下沉量」備援。站遠一點、讓鏡頭照到膝蓋以上最準。
- 填土：兩手腕都明顯高過肩才算（舉手優先於蹲下，避免同時觸發）。

## 📺 投影到電視

和其他遊戲相同：手機用 Google Home App「投放我的螢幕」到 Chromecast（**建議 5GHz Wi-Fi** 才順），手機橫放電視上方、鏡頭朝玩的人，選「🏃 用身體玩」。

## 技術備註

- 單一 HTML 檔，無需安裝；人體偵測 MediaPipe Pose Landmarker（影像只在裝置本機運算，不上傳）
- 鏡頭功能需要 HTTPS 網址或 localhost
- 本遊戲以**橫式**設計（電視/手機橫放），直拿手機會出現轉向提示
