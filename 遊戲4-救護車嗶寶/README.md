# 🚑 救護車嗶寶（遊戲 4）

體感跑酷遊戲（橫式設計）。你是救護車**嗶寶**，在馬路上一路向前衝、左右換車道，吃糖果愛心、閃開三角錐，看到受傷的小車車就開過去**救援**！集滿能量還能**變身機器人**衝刺！🤖

> 原創角色，靈感來自小朋友愛玩的救護車跑酷投幣機，但角色造型、名稱、玩法配置全部自己重做，不使用任何他人版權素材。

## 畫面（真 3D 卡通跑道）

用 Three.js 即時 3D 渲染（toon 卡通著色）：圓潤丘陵、低多邊形樹、路燈、景深霧、漸層天空與太陽，第三人稱鏡頭跟在嗶寶後方。車道由遠方往鏡頭衝來，三線道左右換道。

## 玩法

- **左右移動身體** → 嗶寶在三條車道間左右換道
- **吃收集物加分**：🍬糖果 / ❤️愛心（雙倍分）/ 🎈氣球 / ⭐ / 🍭 / 🧁
- **閃開障礙**：🚧三角錐 / 🪨石頭。撞到會**整台翻滾轉圈圈、冒星星** 😵💫 超好笑（不會結束遊戲，翻完自己爬起來，還有 2 秒無敵）
- **🚑 救援（嗶寶的專長）**：路上出現 🤕 受傷的小車車（頭上閃 🆘），開到牠那條車道把牠救起來 → 治療鈴聲 ✚❤️、放煙火、加 50 分！沒救到也沒關係，牠會自己過去。
- **🧲 磁鐵道具**：吃到後 6 秒內自動把附近糖果吸過來
- **🤖 變身機器人**：吃東西/救人累積「變身能量」，集滿後**雙手高舉**（手指模式點畫面上方）→ 變身機器人衝刺 6 秒，**無敵、撞飛所有障礙、速度加倍**！
- 距離（KM）會一直累積、速度越來越快，但**沒有 Game Over、沒有失敗** ✨
- **🛑 護眼提醒**：跑太靠近電視 → 警報音 + 遊戲強制暫停 5 秒（畫面倒數），退後才能繼續玩

## 手指模式

拖曳左右換車道；**點畫面上方**＝變身（能量滿時）。適合在電腦或平板上玩。

## 體感備註

- 左右換道：用兩肩中點的水平位置判定，身體往左／往右站即可。
- 變身：兩手腕都明顯高過頭才算（高門檻，避免亂揮誤觸）。

## 📺 投影到電視

和其他遊戲相同：手機用 Google Home App「投放我的螢幕」到 Chromecast（**建議 5GHz Wi-Fi** 才順），手機橫放電視上方、鏡頭朝玩的人，選「🏃 用身體玩」。

## 技術備註

- 單一 HTML 檔，無需安裝；3D 畫面用 Three.js（CDN 載入），人體偵測 MediaPipe Pose Landmarker（影像只在裝置本機運算，不上傳）
- 鏡頭功能需要 HTTPS 網址或 localhost
- 本遊戲以**橫式**設計（電視/手機橫放），直拿手機會出現轉向提示
